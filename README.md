## Dev

I recommend use `pnpm` to manage package.

### Initialization

```shell
pnpm i
```

### Run dev

```shell
pnpm dev
```

### Build

```shell
pnpm build
```

### Deploy

```shell
bash ../.github/workflows/deploy.sh
```

> note: You need change username to yours.
